Clonar repositorio en 'Git_Workspace'
git clone (link de clonacion)

Entrar a la carpeta del repositorio
cd (Nombre del Proyecto)

Ver estados de los archivos
git status

Agregar el archivo al repositorio
git add (Nombre archivo)

Comentar un archivo
git commit -m "(Comentario)" (Nombre del archivo)

Agregar el archivo a la direccion master
git push -u origin master
